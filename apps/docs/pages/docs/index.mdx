# Introduction

Welcome to the VitNode documentation! Here you will find all the information you need to get started with VitNode.

## What is VitNode?

VitNode is a modern, open-source, and self-hosted content management system (CMS) that allows you to create and manage your website with ease using NextJS & NestJS.

## Installation

### Requirements

- [Node.js 20+](https://nodejs.org/)
- [PostgreSQL 16+](https://www.postgresql.org/)

### Automatic Installation

We're recommended to use the automatic installation method. You can run the following command to install VitNode.

```bash
npx create-vitnode-app my-app
```

## User Requirements

VitNode has some minimum and recommended requirements for hardware and software. Below you can find the requirements for running VitNode correctly.

### 🖥️ Hardware

|         | Minimum        | Recommended    |
| :------ | :------------- | :------------- |
| CPU     | 1GHz (2+ core) | 1GHz (4+ core) |
| RAM     | 2GB            | 4GB            |
| Storage | 10GB           | 20GB           |
| Network | 100Mbps        | 300Mbps        |

### 🧑‍💻 Browser

|         | Minimum | Recommended |
| :------ | :------ | :---------- |

| Firefox | >= 110  | >= 121      |
| Edge    | >= 106  | >= 120      |
| Safari  | >= 16   | >= 17.2     |
