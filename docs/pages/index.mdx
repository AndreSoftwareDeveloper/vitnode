## What is VitNode?

VitNode is a headless CMS built with [NextJS](https://nextjs.org/) and [NestJS](https://nestjs.com/). It is designed to be a scalable and flexible platform for creating community-driven websites and applications. VitNode uses [Tailwind CSS](https://tailwindcss.com/) and [Shadcn UI](https://ui.shadcn.com/) for its frontend styling, which makes it easy to customize the look and feel of the website.

## Main Features

- **Content management system**: VitNode provides a user-friendly content management system for managing your website's content. You can create, edit, and delete pages, posts, and other content types with ease.
- **User management**: VitNode allows you to create and manage users for your website. You can assign different roles and permissions to users, so they can only access the content that they are authorized to see.
- **Authentication** and authorization: VitNode supports a variety of authentication and authorization methods, so you can secure your website and protect your content.
- **Scalability**: VitNode is designed to be scalable, so you can easily add new features and content to your website as it grows.
- **Flexibility**: VitNode is a flexible platform, so you can use it to create any type of website or application.
