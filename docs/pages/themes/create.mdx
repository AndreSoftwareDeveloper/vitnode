# Create Theme

VitNode allows you to create your own theme which allows user to change the look of the page.

<Callout type="warning" emoji="⚠️">
  Make sure you have [developer mode](/dev) enabled.
</Callout>

## Plugin schema

When you're created a theme VitNode will clone files form the default theme.

<FileTree>
  <FileTree.Folder name="frontend" defaultOpen>
    <FileTree.Folder name="themes" defaultOpen>
      <FileTree.Folder name="{id_theme}" defaultOpen>
        <FileTree.Folder name="core" />
        <FileTree.Folder name="forums" />
        <FileTree.Folder name="...other plugins" />
        <FileTree.File name="theme.json" />
      </FileTree.Folder>
    </FileTree.Folder>
  </FileTree.Folder>
</FileTree>
