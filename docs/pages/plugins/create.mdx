# Create Plugin

In VitNode we're implementing a plugin system. This means that you can create your own plugins and use them in your VitNode instance.

<Callout type="warning" emoji="⚠️">
  Make sure you have [developer mode](/dev) enabled.
</Callout>

## Plugin schema

When you're created a plugin VitNode will created few files for you. Below you can see the schema of the plugin.

  <FileTree.Folder name="backend" defaultOpen>
