# Notifications Toast

VitNode is integrated with [sonner](https://github.com/emilkowalski/sonner) liblary to provide toast notifications created by [Emil Kowalski](https://github.com/emilkowalski).

## Usage

```tsx
import { Toaster, toast } from 'sonner';

export const Example = () => {
  return <button onClick={() => toast('My first toast')}>Give me a toast</button>;
};
```

## API

Below is the list of available methods supported by VitNode.

### Tabs

import { Tabs } from 'nextra/components';
