# Avatar

To use the Avatar component you have to import from `@/components/user/avatar/avatar-user`.
Component requires `sizeInRem` prop to be passed.

<Callout type="info" emoji="ℹ️">
  1rem === 16px
</Callout>

## Current User

If you want to display the current user's avatar, you don't need to pass any props _(except for `sizeInRem` if you want to change the size)_.

```tsx
import { AvatarUser } from '@/components/user/avatar/avatar-user';

<AvatarUser sizeInRem={1.75} />;
```

## Other Users

If you want to display other user's avatar, you have to pass `user` prop with:

- `id` - user's id
- `name` - user's name
- `avatar` - user's AvatarObj interface,

```tsx
<AvatarUser
  user={{
    avatar,
    name,
    id
  }}
  sizeInRem={1.75}
/>
```
