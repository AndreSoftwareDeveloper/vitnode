# Internationalization (i18n)

VitNode uses not only native i18n form [next-intl](https://github.com/amannn/next-intl), but also translations from database. It gives your full control for your translations without restart your server.

Each translations are stored in database as `array`.

```ts
[
  {
    language_code: 'en',
    value: 'Hello world'
  },
  {
    language_code: 'pl',
    value: 'Witaj Å›wiecie'
  }
];
```

## Display text

To display text you can use `useTextLang` hook.

```tsx
import { useTextLang } from '@/hooks/core/use-text-lang';

const { convertText } = useTextLang();

const text = convertText(value);
```

`value` is `TextLanguage[]` interface form `GraphQL backend`.

## Form

Create form with translations is very similar to normal [Forms](/frontend/forms).

<Steps>

### Define schema

```ts
import * as z from 'zod';

const formSchema = z.object({
  name: z.array(
    z.object({
      language_code: z.string(),
      value: z.string()
    })
  )
});
```

You can add more functions to validate translations.

```ts {8, 11}
import * as z from 'zod';

const formSchema = z.object({
  name: z
    .array(
      z.object({
        language_code: z.string(),
        value: z.string().min(3).max(50)
      })
    )
    .min(1)
});
```

### Set initial values

```ts {7}
import { useForm } from 'react-hook-form';
import { zodResolver } from '@hookform/resolvers/zod';

const form = useForm<z.infer<typeof formSchema>>({
  resolver: zodResolver(formSchema),
  defaultValues: {
    name: []
  }
});
```

### Create form field

```tsx {10}
<Form {...form}>
  <form onSubmit={form.handleSubmit(onSubmit)} className="space-y-4">
    <FormField
      control={form.control}
      name="name"
      render={({ field }) => (
        <FormItem>
          <FormLabel>{t('name.label')}</FormLabel>
          <FormControl>
            <TextLanguageInput {...field} />
          </FormControl>
          <FormMessage />
        </FormItem>
      )}
    />

    <Button disabled={!form.formState.isValid} loading={form.formState.isSubmitting} type="submit">
      {tCore('create')}
    </Button>
  </form>
</Form>
```
